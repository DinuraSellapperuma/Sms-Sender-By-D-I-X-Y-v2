== Description ==
SMS Sender by Dixy allows you to easily send SMS notifications directly from your WordPress website. Whether you're sending out order updates, marketing messages, or alerts, this plugin simplifies the process by integrating with your preferred SMS gateway.

== Installation ==
1. Upload the `sms-sender-by-dixy.zip` file through the WordPress plugin manager or extract it and upload the `sms-sender-by-dixy` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the settings page to configure your SMS gateway and other settings.

== Frequently Asked Questions ==

= What SMS gateways are supported? =
Richmo sms Gateway or D I X Y sms Gateway (Only Work with Sri Lankan Numbers)

= Can I customize the SMS content? =
Yes, you can customize the content of the SMS messages through the plugin's settings.

= Is this plugin free to use? =
Yes, SMS Sender by Dixy is free to use. However, depending on your SMS gateway, sending SMS messages may incur costs.
